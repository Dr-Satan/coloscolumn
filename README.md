# Coloscolumn

> **A button to rule them all!** 👑

A Vue 3 application built with Vite, showcasing the legendary **ColosButton** — the one button to rule them all.

## Requirements

- [Node.js](https://nodejs.org/) v18 or later
- npm v9 or later (included with Node.js)

## Getting started

```bash
# Install dependencies
npm install

# Start the dev server (http://localhost:5173)
npm run dev
```

## Available scripts

| Script | Description |
|---|---|
| `npm run dev` | Start the Vite development server with hot-reload |
| `npm run build` | Build for production (output in `dist/`) |
| `npm run preview` | Preview the production build locally |

## Project structure

```
coloscolumn/
├── index.html              # HTML entry point
├── vite.config.js          # Vite configuration
├── package.json
└── src/
    ├── main.js             # App bootstrap
    ├── App.vue             # Root component
    ├── assets/
    │   └── main.css
    └── components/
        └── ColosButton.vue # The button to rule them all
```

## Tech stack

- [Vue 3](https://vuejs.org/) – Composition API (`<script setup>`)
- [Vite](https://vitejs.dev/) – Lightning-fast dev server & bundler
